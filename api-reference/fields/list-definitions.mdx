---
title: 'List Field Definitions'
api: 'GET /api/v1/fields/definitions'
description: 'Get all field definitions for an entity type'
---

## Query Parameters

<ParamField query="entityType" type="string" required>
  Entity type to get definitions for. Options: `company`, `account`, `contact`
</ParamField>

<RequestExample>
```bash cURL
curl -X GET 'https://api.arali.ai/api/v1/fields/definitions?entityType=company' \
  -H 'Authorization: Bearer YOUR_API_TOKEN'
```

```javascript Node.js
const response = await fetch('https://api.arali.ai/api/v1/fields/definitions?entityType=company', {
  headers: {
    'Authorization': 'Bearer YOUR_API_TOKEN'
  }
});

const data = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://api.arali.ai/api/v1/fields/definitions',
    params={'entityType': 'company'},
    headers={
        'Authorization': 'Bearer YOUR_API_TOKEN'
    }
)
```
</RequestExample>
