<RequestExample>
```bash cURL
curl -X GET 'https://api.arali.ai/api/v1/meetings' \
  -H 'Authorization: Bearer YOUR_API_TOKEN'
```

```javascript Node.js
const response = await fetch('https://api.arali.ai/api/v1/meetings', {
  headers: {
    'Authorization': 'Bearer YOUR_API_TOKEN'
  }
});

const data = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://api.arali.ai/api/v1/meetings',
    headers={
        'Authorization': 'Bearer YOUR_API_TOKEN'
    }
)
```
