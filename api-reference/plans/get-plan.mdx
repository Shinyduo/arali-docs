---
title: 'Get Plan'
api: 'GET /api/v1/plans/{key}'
description: 'Retrieve a specific plan by key'
---

## Path Parameters

<ParamField path="key" type="string" required>
  The unique key of the plan to retrieve
</ParamField>

## Example Request

```bash
curl -X GET https://api.arali.ai/api/v1/plans/{key} \
  -H "Authorization: Bearer YOUR_API_TOKEN"
```

## Response

Returns the plan record with full details including:
- Plan ID
- Key
- Name
- Description
- Status
- Pricing configuration
- Associated products
- Created and updated timestamps
