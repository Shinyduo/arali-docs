---
title: 'Remove Product from Plan'
api: 'DELETE /api/v1/plans/{planId}/products/{productKey}'
description: 'Remove a product from a specific plan'
---

## Path Parameters

<ParamField path="planId" type="string" required>
  The unique ID of the plan
</ParamField>

<ParamField path="productKey" type="string" required>
  The key of the product to remove from the plan
</ParamField>

## Example Request

```bash
curl -X DELETE https://api.arali.ai/api/v1/plans/{planId}/products/{productKey} \
  -H "Authorization: Bearer YOUR_API_TOKEN"
```

## Response

Returns a success confirmation upon removal.
