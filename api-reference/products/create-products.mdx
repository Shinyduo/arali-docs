---
title: 'Create/Update Products'
api: 'POST /api/v1/products'
description: 'Batch create or update product records'
---

## Request Body

<ParamField body="products" type="array" required>
  Array of product objects to create or update
  
  <Expandable title="Product Object">
    <ParamField body="key" type="string" required>
      Unique identifier for the product (e.g., "image-analysis")
    </ParamField>
    
    <ParamField body="name" type="string" required>
      Display name of the product
    </ParamField>
    
    <ParamField body="description" type="string">
      Detailed description of the product
    </ParamField>
    
    <ParamField body="kind" type="string" required>
      Product type: `core_product`, `feature`, `addon`, or `metric_only`
    </ParamField>
    
    <ParamField body="status" type="string" required>
      Product status: `active` or `archived`
    </ParamField>
    
    <ParamField body="isBillableCandidate" type="boolean">
      Whether this product can be billed. Default: `false`
    </ParamField>
  </Expandable>
</ParamField>

## Example Request

```bash
curl -X POST https://api.arali.ai/api/v1/products \
  -H "Authorization: Bearer YOUR_API_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "products": [
      {
        "key": "image-analysis",
        "name": "Image Analysis",
        "description": "AI-powered image analysis service",
        "kind": "core_product",
        "status": "active",
        "isBillableCandidate": true
      },
      {
        "key": "advanced-search",
        "name": "Advanced Search",
        "description": "Enhanced search capabilities",
        "kind": "feature",
        "status": "active",
        "isBillableCandidate": true
      }
    ]
  }'
```

## Response

Returns the created/updated product records with their generated IDs.
