---
title: 'Get Product'
api: 'GET /api/v1/products/{productId}'
description: 'Retrieve a specific product by ID'
---

## Path Parameters

<ParamField path="productId" type="string" required>
  The unique ID of the product to retrieve
</ParamField>

## Example Request

```bash
curl -X GET https://api.arali.ai/api/v1/products/{productId} \
  -H "Authorization: Bearer YOUR_API_TOKEN"
```

## Response

Returns the product record with full details including:
- Product ID
- Key
- Name
- Description
- Kind
- Status
- Billable candidate status
- Created and updated timestamps
