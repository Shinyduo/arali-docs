---
title: 'Report Usage'
api: 'POST /api/v1/usage'
description: 'Batch report usage data for products'
---

## Request Body

<ParamField body="usage" type="array" required>
  Array of usage records to report
  
  <Expandable title="Usage Record">
    <ParamField body="externalCompanyId" type="string" required>
      External ID of the company
    </ParamField>
    
    <ParamField body="externalAccountId" type="string">
      External ID of the account (optional)
    </ParamField>
    
    <ParamField body="productKey" type="string" required>
      The key of the product being used
    </ParamField>
    
    <ParamField body="metricKey" type="string" required>
      The key of the metric being tracked (e.g., "api_calls", "search_queries")
    </ParamField>
    
    <ParamField body="date" type="string" required>
      Date of usage in YYYY-MM-DD format
    </ParamField>
    
    <ParamField body="value" type="number" required>
      Usage value for the metric
    </ParamField>
  </Expandable>
</ParamField>

## Example Request

```bash
curl -X POST https://api.arali.ai/api/v1/usage \
  -H "Authorization: Bearer YOUR_API_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "usage": [
      {
        "externalCompanyId": "acme-corp",
        "externalAccountId": "acme-enterprise-account",
        "productKey": "image-analysis",
        "metricKey": "api_calls",
        "date": "2024-01-15",
        "value": 1500
      },
      {
        "externalCompanyId": "acme-corp",
        "externalAccountId": "acme-enterprise-account",
        "productKey": "advanced-search",
        "metricKey": "search_queries",
        "date": "2024-01-15",
        "value": 320
      }
    ]
  }'
```

## Response

Returns the created usage records with their IDs.
