## Core Features

<AccordionGroup>
  <Accordion title="Companies & Accounts" icon="building">
    Manage company records and accounts linked to your enterprise. Track health scores, ARR, lifecycle stages, and custom fields.
  </Accordion>
  <Accordion title="Contacts" icon="users">
    Create and manage contact records with multiple emails, phone numbers, and company associations.
  </Accordion>
  <Accordion title="Interactions" icon="phone">
    Track calls and meetings, including recordings, participants, and metadata.
  </Accordion>
  <Accordion title="Custom Fields" icon="sliders">
    Define custom field schemas and store custom data for any entity type.
  </Accordion>
</AccordionGroup>

## Base URL

All API requests should be made to:

```
https://api.arali.ai
```

## API Version

The current API version is `v1`. All endpoints are prefixed with `/api/v1`.

<Note>
  This documentation covers the public API endpoints. For webhook integrations (Aircall, HubSpot, Slack, WhatsApp), please contact support.
</Note>
